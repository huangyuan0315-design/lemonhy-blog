## 关于这个博客

欢迎来到我的个人博客！这个博客使用以下技术构建：

- **Astro 7** — 静态网站生成器，零 JS 默认输出
- **Tailwind CSS v4** — 实用优先的 CSS 框架
- **Cloudflare Pages** — 全球 CDN 托管
- **Cloudflare Workers + D1** — 评论系统后端
